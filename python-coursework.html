Project 2: Scuba Expedition Application

Description: This application calculates the total cost for a scuba diving expedition based on location (Australia, Belize, or Fiji), the specific dive selected, and the number of divers. The program uses a custom function to eliminate code repetition and handle dive selection, validation, and cost calculation efficiently.

Code:

'''
Application Name: Scuba Expedition Application
Developer: Graham Gobbel
Description: The application calculates the total due for a scuba expedition based on the selected location, dive selected, and the number of people in the group. 
Date: 2/24/26
'''

####################### New Code for Assignment 4 ###############################
#adding a function below to simplify the code and get rid of the repetition/redundancy that happens with all of the if statements when selecting a dive. This function 
# takes in the dive options, location, and number of divers, and then prints the available dives, validates the user's selection, and calculates the total cost
def choose_dive(dives, location, number_divers):
    print(f"Available dives in {location}:")

    for dive in dives: #for loop that prints the dives in the chosen location and their costs
        print(dive, ":", "$", dives[dive])

    selected_dive = input("What dive are you selecting? ") #ask user to select a dive
    while selected_dive not in dives:  #makes sure that the dive chosen is one of the options available for that location, if not ask the user to re-enter a dive
        print("Please select a dive from the options above.")
        selected_dive = input("What dive are you selecting? ")

    total_cost = number_divers * dives[selected_dive] #calculates the total cost by multiplying the number of divers by the cost of the selected dive

    return selected_dive, total_cost #the output is the selected dive and the total cost, which will be printed at the end of the program
#################################################################################

if __name__ == "__main__":

    #create dictioniaries for each location right off the bat for simplicity
    Australia= {'Ribbon Reef': 650, 'Byron Bay': 400}

    while location not in valid_locations: #validate that the location is one of the ones listed
        print("Please select a country from the above options")
        location = input("The diving location options are Australia, Belize, and Fiji.\nWhat location are you diving in?") #if user entered an invalid loocation, ask the to re-enter the location

    number_divers= int(input("How many people are in your dive team? ")) #user inputs the amount of people in their dive team
    while number_divers <=0:
        print("Please enter a dive team size greater than 0.") #validate that the number of divers is a positive integer, if not ask the user to re-enter the number of divers
        number_divers= int(input("How many people are in your dive team? "))

    ######################## New Code ##############################

    #Removed old if statements and replaced with simpler if statements. These call the new function which takes care of printing the dive options, validating the user's selection, 
    #and calculating the total cost. This makes the code much cleaner and easier to read, while still achieving the same functionality.

    if location == "Australia": #if the user types in Australia, the function will be called with the Australia dive options, location, and number of divers
        selected_dive, total_cost = choose_dive(Australia, location, number_divers)

        selected_dive, total_cost = choose_dive(Belize, location, number_divers)

    print(f"You have selected {selected_dive} dive in {location} for {number_divers} people at a total cost of ${total_cost:,}. Thank you!")

Project 3: Data Visualization with Matplotlib

Description: This application creates a horizontal bar graph to visualize categorical and numerical data using the matplotlib library. Users input a graph title, column headers, and data points, which are then displayed as an interactive horizontal bar chart.

Code:

'''
Application Title: VS Code Assignment 5
Developer: Graham Gobbel
Date: 3/3/2026
Purpose: Display a horizontal bar graph of categorical and interger data using the matplotlib library.
'''

import matplotlib.pyplot as plt # imports a matplot library to do plotting 

#prompt the user for a title for the graph
from ast import While


graph_title= input("Enter title for the data:")
print("You entered: ", graph_title)

#Prompt the user for the headers of two columns of a table. Output the column headers
column1_header = input("Enter the column 1 header: \n")
print("You entered:", column1_header)
column2_header = input("Enter the column 2 header: \n")
print("You entered:", column2_header)

#Prompt the user for data points
# the user should enter a string and an integer separated by a comma. 
#Store the string components of the data points in a list of strings. 
#Store the integer components of the data points in a list of integers

#variable intialization
data_entry= ''
data_categories = []
data_numbers = []
data_entry= input("Enter a data point (-1 to stop input): ")
while data_entry != '-1':
    my_tokens= data_entry.split(',')
    data_categories.append(my_tokens[0])
    data_numbers.append(int(my_tokens[1]))
    print('Data string: ', my_tokens[0])
    print('Data integer: ', my_tokens[1])
    data_entry= input("Enter a data point (-1 to stop input): ")


objects = data_categories
x_pos = range(len(objects)) # creates the number of bars required
performance = data_numbers

plt.barh(x_pos, performance)  #creates a horizontal bar graph ploting performance versus the number of bars required
plt.yticks(x_pos, objects)    #labels the bars with programming language names
plt.xlabel(column2_header)
plt.title(graph_title)
plt.show()
